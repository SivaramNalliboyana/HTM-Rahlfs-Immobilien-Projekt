How the AI decides

Four levels. Two to four actions. One deadline each.

Kritisch Sicherheit — Gas, Strom, Brand, akute Wassergefahr. Sofort-Maßnahme.
Hoch Bewohnbarkeit eingeschränkt — kein Wasser, keine Heizung im Winter, kein Strom.
Mittel Beeinträchtigung, Wohnung nutzbar — begrenzter Wasserschaden, schwache Heizung.
Niedrig Kosmetisch oder geringfügig — Lackschaden, lockerer Beschlag.

Plus 2–4 Aktionen mit Frist.

Every triaged case carries concrete next steps for the manager — written in the same prompt, persisted once, never re-generated.

What we tested this week

Build · Measure · Learn — one cycle in seven days.

Built

Telegram → Cockpit → PDF

End-to-end MVP. Python bot, FastAPI triage, React cockpit, client-side PDF. JSON on disk on purpose — keep the loop tight.

Measured

10 simulated intakes

Real Mängel templates from the Rahlfs archive. Median tenant time: 4 min. Median manager review time before action: under 30 sec.

Learned

9/10 triage match

AI urgency matched the senior manager's call on 9 cases. The miss was the boundary of "akute Wassergefahr" — prompt fixed, re-tested.

Next test
